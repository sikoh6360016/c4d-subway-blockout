# C4D Subway Blockout Script

Cinema 4D Python script that generates a futuristic subway / escalator blockout scene.

## Features
- Automatic blockout generation
- Wide cinematic camera setup
- Editable primitive-based structure
- Designed for AI + 3D workflow

## Usage
Run inside Cinema 4D Script Manager.

## Workflow
GPT → Codex → C4D → AI rendering
